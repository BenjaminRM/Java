Backtracking lab - Lab10 from CS for Transfers course at RIT (CSAPX).  Lab completed Fall Semester of 2016-2017 year.
I was supplied with Backtracker.java, Configuration.java, TentsAndTrees.java, and I was assigned to create TentConfig.java
to complete the assignment. See folder "ProjectFiles/src/" for the .java files.

USAGE:
	$java -jar Lab10.jar <OPTION1> <OPTION2>

Where OPTION1 is:
	data/tents1.txt
	data/tents2.txt
	data/tents3.txt
	data/tents4.txt
	data/tents5.txt
	data/tents6.txt
	data/tents7.txt
	data/tents8.txt

Higher the tent file #, the more difficult the problem.

Where OPTION2 is:
	true
	false

Where:
	true = visual debugger enabled (really cool)
	false = visual debugger disabled (not so cool)

Example Runs:
	java -jar Lab10.jar data/tents1.txt true
	java -jar Lab10.jar data/tents8.txt true
	java -jar Lab10.jar data/tents4.txt false
